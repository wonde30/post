# Post

A full-stack web application built with Laravel 13 and Vue 3.

## Tech Stack

### Backend

* Laravel 13
* Sanctum
* MySQL

### Frontend

* Vue 3
* Vite
* Pinia
* Axios
* Tailwind CSS 4

## Project Structure

```text
post
├── client
└── server
```

## Setup

### Backend

```bash
cd server
composer install
php artisan serve
```

### Frontend

```bash
cd client
npm install
npm run dev
```
